The Patterson Camp

A simple website built to showcase a rental property in the state of Maine. Users are able to view pictures of the campsite and view a calendar that shows availabile and booked days.

Technology

The Patterson Camp website uses vanilla HTML, CSS, and JS to create the design and is responsive on both mobile and tablets. I used a css trick to create hamburger menu button on mobile.
